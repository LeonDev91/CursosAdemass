Introducción a JavaScript

Qué es JavaScript

Es un lenguaje de programación, interpretado, orientado a objetos, imperativo, case sensitive, basado en prototipos/instancias/classless, de tipado débil, y es dinámico.

Para qué se usa

Dinamismo en sitios web, servidor en NodeJS, tecnologías frontend como Angular, React o Vue.JS

Otros usos no tan comunes: inteligencia artificial, placas electrónicas (mediante Johnny Five), mobile apps, desktop apps, etc.

Formas de enlazar JavaScript

  • En línea
  • Como contenido en la etiqueta "script"
  • Como contenido en un archivo de formato .js
  • Con un require
