Conexión y Selectores

Existen 3 formas de conectar nuestros estilos al documento html:

  • En línea: Se realiza mediante el atributo style dentro de la etiqueta a la cual se desea darle estilo.
  • En cabecera: Mediante la etiqueta style anidada en la etiqueta head.
  • Externo: Se realiza mediante la etiqueta link anindada en la etiqueta head y su href será la ruta del documento css a enlazar. Esta forma de enlazar el css y el html nos permite reutilizar los estilos sin necesidad de editar el documento html.
